### On rendering graphs 
A good option to explore would be [React Three Fiber](https://r3f.docs.pmnd.rs/getting-started/introduction).
    - Provides [good options for 3d rendering](https://threejs.org/docs/index.html). The tutorial recommends keeping tabs open on 
        1. Scene
        2. Camera.
        3. Mesh.
        4. Material. 
        5. Tutorial doesn't talk about this, but I think that Sprite would be better than mesh. They could just be little dots with random (preset) colors. (UH OH, GRAPH COLORING PROBLEM INCOMING) 
    - Another thought: it might be good to find "arms" of the graph using [Fermat's Spiral](https://en.wikipedia.org/wiki/Fermat%27s_spiral). This basically is a fractal-like pattern 
        - The representation would use polar coordinates. Conversions to cartesian coordinates use pretty standard conversion formulae that are found on the [Wikipedia page](https://en.wikipedia.org/wiki/Fermat's_spiral#Cartesian) linked.
